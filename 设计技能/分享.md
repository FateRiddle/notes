## 团队组织

1. 团队（特别是一线人员） 现状和挑战

- 现状 “战略-策略-执行”理解不深，个体和全局的关联感不强

2. 人才梯队

- 头狼梯队 当下急缺中场发动机和技术、业务域的一号位

3. 对 TL 和团队头狼要求

- KPI 透明， code review， 京杭定期互通， 代码透明， Aone 事物透明

4. 对研发的要求

- PM 制度： owner 要有清晰的目标和路线图，信息透明，开会高质量
